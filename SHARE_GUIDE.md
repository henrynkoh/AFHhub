## Quick Setup Instructions for Partners

### Prerequisites
- Node.js 18+ installed
- Git installed
- Modern web browser

### Setup Steps
```bash
# 1. Clone the repository
git clone https://github.com/henrynkoh/AFHhub.git

# 2. Navigate to project
cd AFHhub

# 3. Install dependencies
npm install

# 4. Start development server
npm run dev

# 5. Open in browser
# Navigate to http://localhost:3000
```

### Troubleshooting
- If `npm install` fails, try `npm install --legacy-peer-deps`
- If port 3000 is busy, Next.js will suggest another port
- Clear browser cache if seeing old version
