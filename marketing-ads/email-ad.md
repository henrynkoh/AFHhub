## Email Design Guidelines

**Layout:**
- Single column design
- Mobile-responsive
- Clear hierarchy
- Ample white space
- Consistent branding

**Colors:**
- Primary: Blue (#3B82F6)
- Secondary: Purple (#8B5CF6)
- Text: Dark gray (#1F2937)
- Background: White (#FFFFFF)
- Links: Blue, underlined

**Typography:**
- Headings: Bold, 20-24px
- Body: Regular, 16px
- Buttons: Bold, 16-18px
- Line height: 1.5-1.6

**Buttons:**
- Prominent, clear call-to-action
- Contrasting color
- Rounded corners
- Adequate padding
- Hover effects

**Images:**
- Optimized file size
- Alt text for accessibility
- Responsive sizing
- Relevant to content

## Best Practices

**Subject Lines:**
- Keep under 50 characters
- Include value proposition
- Create urgency when appropriate
- Test different variations
- Avoid spam trigger words

**Content:**
- Keep it concise
- Use bullet points
- Include clear call-to-action
- Personalize when possible
- Provide value

**Timing:**
- Send during business hours
- Test different days/times
- Consider time zones
- Avoid holidays
- Monitor engagement

**Testing:**
- Test on multiple email clients
- Check mobile rendering
- Verify all links work
- Test images load properly
- Check spam score
