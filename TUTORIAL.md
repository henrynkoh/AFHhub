**Practice:**

### Step 3: Understanding the Matching Algorithm

**Action:**
1. Read about the matching criteria:
   - Care level compatibility (30% weight)
   - Payment type acceptance (25% weight)
   - Availability and waitlist (15% weight)
   - Location proximity (15% weight)
   - Specialty match (10% weight)
   - Response time (5% weight)
2. Understand how scores are calculated
3. Learn about additional factors:
   - +5 points for highly rated providers (4.5+)
   - -10 points for long waitlists (>10)

**Matching Process:**
1. System analyzes all providers
2. Calculates compatibility scores
3. Applies weighted criteria
4. Adds bonus/penalty points
5. Sorts by score (highest first)
6. Returns top 10 matches

**Score Interpretation:**
- 90-100%: Excellent match
- 80-89%: Very good match
- 70-79%: Good match
- 60-69%: Acceptable match
- Below 60%: Poor match

**Practice:**
- Understand each criterion
- Calculate sample scores
- Review match reasoning
- Learn score interpretation
