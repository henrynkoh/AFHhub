## üîç AFH Provider Search - Quick Start

### Find Providers in 3 Steps

**Step 1: Search**
- Type a city name (e.g., "Centralia") in the search bar
- Or click quick filter buttons (Centralia, Medicaid, LLC)
- Results update instantly as you type

**Step 2: Review Results**
- See provider information in the table
- Click column headers to sort
- Review: Location, Name, Phone, Experience, Capacity, Payment Types

**Step 3: Contact**
- Mobile: Tap phone number to call directly
- Desktop: Right-click to copy number or visit website
- Always verify information is current

### Quick Search Tips
- **By Location**: Type city name
- **By Payment**: Type "Medicaid" or "Private Pay"
- **By Phone**: Type area code (e.g., "360")
- **Combined**: "Centralia Medicaid" finds Medicaid providers in Centralia

## üè• AFH Resident Placement Services - Quick Start

### Place a Resident in 4 Steps

**Step 1: Add Resident**
- Click "Add New Resident"
- Fill required fields: Name, DOB, Location, Care Level, Payment Type, Priority
- Add optional: Medical conditions, special needs
- Submit form

**Step 2: Generate Matches**
- Select resident from list
- Click "Generate Matches"
- System analyzes all providers
- Returns top 10 matches with scores (0-100%)

**Step 3: Review Matches**
- Check match scores (90%+ = excellent)
- Read detailed reasoning for each match
- Compare multiple matches
- Verify provider availability

**Step 4: Accept & Place**
- Click "Accept Match" for chosen provider
- Contact provider to confirm
- Proceed with placement process
- Update status to "Placed" when complete

### Quick Placement Tips
- Set priority level correctly (Urgent, High, Medium, Low)
- Review all match reasoning before deciding
- Contact providers to verify availability
- Track status regularly for multiple residents
